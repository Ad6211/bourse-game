# 📈 Bourse Game - Beta

Jeu de bourse virtuel en temps réel avec 200 actions et système de trading complet.

## 🎮 Fonctionnalités

### Core Gameplay
- ✅ **200 actions** générées aléatoirement (1µ à 1Mdµ)
- ✅ **15 secteurs** différents (tech, finance, pharma, etc.)
- ✅ **Variations en temps réel** chaque seconde
- ✅ **Système d'événements** aléatoires (10% de chance)
- ✅ **Pagination** avec tri intelligent
- ✅ **Formatage automatique** (µ, kµ, Mµ, Mdµ)

### Trading
- ✅ Achat/vente avec quantité personnalisée
- ✅ Suivi du portefeuille avec prix d'achat
- ✅ Indicateurs de profit/perte en temps réel
- ✅ Graphiques d'historique pour chaque action

### Systèmes Avancés
- ✅ **Favoris** (⭐) - Jusqu'à 2+ actions favorites
- ✅ **Alertes de prix** (🔔) - Notifications automatiques
- ✅ **7 améliorations** avec coûts exponentiels
- ✅ **13 achievements** à débloquer
- ✅ **Sauvegarde automatique** (30s)
- ✅ **Revenu passif** avec AutoTrader

### Interface
- ✅ Design moderne et responsive
- ✅ Animations et notifications élégantes
- ✅ Interface adaptative (mobile/tablette/desktop)
- ✅ Mode sombre/gradient coloré

## 🚀 Installation Rapide

### Version Web (PWA)
1. Ouvrez `index.html` dans votre navigateur
2. C'est tout ! Le jeu fonctionne directement

### Version APK (Android)
Consultez `BUILD_APK.md` pour les instructions détaillées.

**Méthode recommandée**: PWABuilder
1. Allez sur https://www.pwabuilder.com/
2. Entrez l'URL de votre site
3. Sélectionnez Android
4. Téléchargez l'APK

## 🎯 Comment Jouer

1. **Démarrage**: Vous commencez avec 10 µ
2. **Achetez**: Sélectionnez une action et achetez au prix bas
3. **Vendez**: Revendez quand le prix monte
4. **Améliorez**: Investissez dans les upgrades pour augmenter vos revenus
5. **Gagnez**: Devenez milliardaire ! 💰

### Conseils
- Achetez quand les prix sont bas (▼)
- Vendez quand les prix montent (▲)
- Utilisez les alertes pour ne pas manquer les opportunités
- Les événements peuvent faire exploser les prix !
