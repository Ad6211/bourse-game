# 🚀 Démarrage Rapide - APK

## Méthode EXPRESS (5 minutes)

### 1. Hébergez vos fichiers
Uploadez tous les fichiers sur **Netlify Drop** (gratuit, pas de compte requis):
1. Allez sur https://app.netlify.com/drop
2. Glissez-déposez le dossier complet
3. Copiez l'URL fournie (ex: https://random-name.netlify.app)

### 2. Générez l'APK
1. Allez sur https://www.pwabuilder.com/
2. Entrez votre URL Netlify
3. Cliquez "Start" → "Android" → "Build my PWA"
4. Téléchargez l'APK

### 3. Installez
- Transférez l'APK sur votre téléphone Android
- Ouvrez et installez
- C'est prêt !

## Fichiers à uploader sur Netlify
```
dossier-bourse-game/
├── index.html
├── style.css
├── game.js
├── manifest.json
├── sw.js
└── icon.png
```

## C'est tout !
L'APK sera générée automatiquement avec:
- ✅ Icône officielle
- ✅ Mode standalone
- ✅ Pas de barre d'adresse
- ✅ Fonctionne hors-ligne
- ✅ Compatible Android 5+
